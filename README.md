# 📜 《翠花史记》 (The Chronicles of Cuihua)

> **“太史公曰：cuihua 者，乃当世之奇人也。其言其行，不可不记。”**

欢迎来到《翠花史记》编纂委员会官方仓库！本项目致力于客观、公正、严肃（且幽默）地记录 **cuihua** 同学的生平事迹、高光时刻、奇闻异事以及经典语录。

本书采用开源协作模式编纂，集众智以成信史。欢迎各路亲朋好友、同窗同好提交 PR，共同完善这部旷世巨作！

---

## 📖 目录结构 (Table of Contents)

本仓库严格遵循司马迁《史记》的体例进行分类，目前包含以下卷宗：

* **👑 [本纪 (Annals)](./docs/benji/)** * 专为 cuihua 设立。以编年体的形式，记录 cuihua 诞生以来的重大历史事件、人生转折点以及具有里程碑意义的时刻。
* **🏘️ [世家 (Hereditary Houses)](./docs/shijia/)**
  * 记录与 cuihua 渊源颇深、影响力巨大的核心圈子成员（如：挚友、室友、导师或重要合作伙伴）的事迹。
* **🧍 [列传 (Biographies)](./docs/liezhuan/)**
  * 记录围绕在 cuihua 身边的各色人物，以及他们与 cuihua 之间发生的可歌可泣（或啼笑皆非）的交集。
* **📊 [表 (Tables)](./docs/biao/)**
  * 以表格形式梳理的干货。例如：`《cuihua 经典语录年表》`、`《cuihua 历年发型变迁考》`、`《cuihua 摸鱼/重大成就时间线》`等。
* **📚 [书 (Treatises)](./docs/shu/)**
  * 记录 cuihua 在特定领域的制度、典章或独特见解。例如：`《干饭书》`（美食见解）、`《学术书》`（研究心得与踩坑指南）、`《熬夜书》`等。

## ✍️ 如何参与编纂 (How to Contribute)

历史由人民书写，本史记支持多人协作！如果你手中掌握了关于 cuihua 的未解之谜、独家黑料或感人故事，欢迎上奏（Submit a Pull Request）：

1. **Fork** 本仓库到你的个人账号（建立你自己的修史分舵）。
2. 创建一个新的分支，命名为 `feature/add-story-xxx`（研墨铺纸）。
3. 在对应的目录下添加或修改 Markdown 文件，详细记载事迹（奋笔疾书）。
4. 提交你的更改 `git commit -m "史官新增：cuihua某年某月某日事迹"`（盖章落款）。
5. 将分支推送到 GitHub `git push origin feature/add-story-xxx`（快马加鞭）。
6. 提交一个 **Pull Request** 给主仓库（上递奏折）。

*太史公（仓库管理员）将在审核“史料”的真实性（或搞笑程度）后，将其 Merge 入正史。*

## ⚖️ 史官守则 & 免责声明 (Disclaimer)

1. **秉笔直书**：本仓库鼓励分享有趣的真实故事，但请把握尺度。严禁捏造过于恶劣的谣言，做到“不虚美，不隐恶”（适度夸张除外）。
2. **保护隐私**：在记录时，请尽量避免泄露极其敏感的个人隐私信息（如身份证号、家庭住址等），我们修的是史记，不是开盒。
3. **最终解释权**：本《史记》的最终解释权归 cuihua 本人及本仓库维护者所有。若当事人提刀赶来要求“焚书坑儒”（删除某段黑历史），史官将酌情妥协（`git reset --hard`）。
